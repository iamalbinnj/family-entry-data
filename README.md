# family-entry-data



### Setup

1. Clone the repo

   ```sh
   git clone https://github.com/iamalbinnj/family-entry-data.git
   ```

2. Go to the project folder

   ```sh
   cd family-entry-data
   ```

3. Set up your `.env` file

   - Duplicate `.env.example` to `.env`

4. install

   ```sh
   npm install
   ```

5. run

   ```sh
   npm start
   ```
   or
   ```sh
   npm run dev
   ```
